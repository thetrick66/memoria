\section{Problema a resolver y objetivos}
El principal objetivo de este proyecto es coordinar el funcionamiento de dos ascensores prototipos para mejorar la eficiencia en el uso de energía y perfeccionar los movimiento de ambos ascensores para que sus recorridos sean lo más óptimos posibles, para ello se usará un sistema maestro/esclavo donde se utilizará un ascensor como maestro, el cual tendrá prioridad de movimientos y llamados sobre el otro ascensor. El otro ascensor será usado como el esclavo, este ascensor esclavo hará los recorridos que el maestro no haga, siempre y cuando sea el recorrido más eficiente y óptimo.\\

Se comenzará automatizando un solo ascensor
