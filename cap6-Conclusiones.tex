\chapter{Conclusiones}\label{chap6}
Después de observar el correcto funcionamiento de la automatización coordinada del sistema de dos ascensores prototipos y de la supervisión HMI a través de Ethernet utilizando la red disponible del departamento de electrónica, se puede dar por concluidos los objetivos propuestos de mejorar los ascensores prototipos para un mejor funcionamiento ya que son usados constantemente en el laboratorio de control industrial.\\

Además se aprendió a usar la comunicación entre los dos PLC's a través de protocolo PPI para poder intercambiar variables y estados para así generar un control coordinado entre ambos ascensores.\\

Por último, la creación de un HMI para supervisión remota vía Ethernet donde se tuvo que empezar configurando y habilitando el módulo de comunicación Ethernet CP243-1 IT del PLC Siemens S7-200, además de generar un servidor OPC local usando el software PC Access y finalizando con el uso del software de adquisición de datos SCADA iFix quien permite la creación de un interfaz gráfico.
\section{Aportes de la memoria, diferencias y/o mejoras}

El principal aporte de está memoria es el mejoramiento de los ascensores prototipo para futuros usos en el laboratorio de control industrial, ya que estos equipos son constantemente usados como experiencia del ramo con el mismo nombre del laboratorio, ya que es un sistema de eventos discretos que sirve para introducir a los alumnos al control por PLC de diferentes sistemas con características similares. Además de entregar seguridad al sistema para no causar problemas durante el primer acercamiento de automatización, por tanto su gran diferencia a otras memorias es el mejoramiento previo de los sistemas antes de comenzar con la memoria para dejar a siguientes generaciones sistemas actualizados y de mayor grado didáctico para aumentar el aprendizaje y familiarización con el mundo industrial.

\section{Trabajos futuros}

Siempre es necesario ir actualizando los sistemas que son usados de manera didáctica para que sea mejor el grado de aprendizaje de los alumnos que usen este sistema, el sistema tiene muchas posibilidades para hacer futuros trabajos como puede ser agregarle un cuarto piso poniendo un sensor de piso extra y otro par de botones como los del segundo piso y así poder darle un grado de complejidad que permita ver con claridad el correcto funcionamiento de la automatización del sistema.\\

Se puede finalizar el objetivo inconcluso de la supervisión remota a través de una página web usando Java Applets y la capacidad del módulo de comunicación CP243-1 IT en su totalidad.\\

Por último, se pueden agregar funciones al ascensor para utilizar los módulos analógicos con los que cuenta el PLC y así poder usar variables digitales y analógicas para obtener un grado de aprendizaje mucho mayor con este único sistema.
